## Image Specifications

- **Format**: WebP (preferred for better performance) or JPG
- **Recommended Size**: 800x600px or similar aspect ratio
- **File Size**: Optimize images for web (under 200KB recommended)

## Fallback

If an image fails to load, the component will automatically display an emoji icon as a fallback.
